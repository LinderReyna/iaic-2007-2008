PROBLEMA DEL LOBO, LA CABRA Y LA COL
 Representación:

Se han utilizado cuatro enteros, uno para cada animal, y otro para el granjero, que tomaran el valor 0 o 1 dependiendo de la orilla en la que se encuentren. (1 para la orilla izquierda, 0 para la orilla derecha ).


 Operadores:

Tenemos cuatro operadores, dependiendo de si cruza el granjero, el lobo, la cabra o la col. La condición necesaria para que pueda aplicarse cada uno de ellos es que el granjero este en la misma orilla que el lobo
o que la cabra o que la col para poder cruzarlos.

Heurística:

La heurística diseñada para este problema es la suma de los valores de cada entero, la cual será 0 cuando todos estén en la orilla derecha.
