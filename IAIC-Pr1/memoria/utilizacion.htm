UTILIZACIÓN DE LA IMPLEMENTACIÓN

Para poder ejecutar el proyecto se debe descomprimir el archivo entregado en la carpeta c:\hlocal\, se entra 

en la dirección IAICp1\dist\ y se ejecuta el archivo .jar. En la carpeta src se puede encontrar el código

fuente y en la carpeta doc la documentación sobre la fuente, que forma parte de la memoria.

Para poder entender cómo funciona el código del proyecto hemos creado un javadoc

 en donde vienen explicadas cada una de las funciones que utilizamos.
